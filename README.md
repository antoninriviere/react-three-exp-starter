# three-exp-starter
3D Experiment starter to build cool stuff :rocket:

### install : 
yarn
### dev :
yarn dev
### prod :
yarn build

## Technologies

* Three.js
* Stylus
* ES6 w/ Babel
* Webpack 2
* StandardJs linting with eslint
* Wagner
* ControlKit
* Orbit controls
