# three-exp-starter
Web experiments starter using : 
* React
* Three.js
* Wagner
* Orbit controls
* controlKit GUI
* Stylus
* Webpack
* ES6 with Babel
* Standard JS linting

### install : 
yarn install
### dev :
yarn run dev
### prod :
yarn run build
